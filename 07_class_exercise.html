Lecture 7 - Classroom Exercise

In this exercise, you are going to practice some CSS code and pseudo-classes. A good aplication of pseudo-classes is to format form inputs, and that is the focus of the exercise today. Apply the following formats by creating appropriate CSS styles in the head tag:

  • Change the heading (h1) text to your favorite color, and its background color for another color that matches well.
  • When mouse hover a form label, change label text to bold.
  • Change the background color to green if the form input is valid.
  • Create a 1px red border with rounded corners, if the form input is invalid.
  • Change the background color to yellow, and set the border to 0px, if the form input has focus.
